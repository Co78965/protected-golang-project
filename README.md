Реализация программы на ЯП Golang с применением подходов защиты ПО (подсчет CRC, шифрование строк), также в программе реализованы методы обнаружения VM и средств отладки. В качестве противодействия дизассемблированию была использована ассемблерная вставка, а также обфускация потока передачи управления через исключение
